# 🖱️ Python Click Macro Tool

Ferramenta open-source em **Python** para automação de cliques em regiões da tela com interface gráfica.

O usuário pode selecionar regiões arrastando o mouse, definir delays entre cliques e executar sequências automatizadas.

Projetado para ser **simples, extensível e colaborativo**, permitindo que qualquer pessoa melhore o projeto.

---

# ✨ Funcionalidades

✔ Selecionar regiões da tela arrastando o mouse  
✔ Adicionar **quantas regiões quiser**  
✔ Definir **delay entre cliques**  
✔ Executar cliques na **ordem configurada**  
✔ Clique **aleatório dentro da região** (evita padrões robóticos)  
✔ Sistema de **loops configuráveis**  
✔ **Pausar / continuar / parar** execução  
✔ **Atalhos globais de teclado**  
✔ **Salvar e carregar perfis JSON**  
✔ Reordenar regiões facilmente  
✔ Interface gráfica simples e intuitiva  

---

# ⌨️ Atalhos

| Tecla | Ação |
|------|------|
| **F6** | Iniciar macro |
| **F7** | Pausar macro |
| **F8** | Continuar macro |
| **ESC** | Parar macro |

⚠️ Em alguns sistemas pode ser necessário executar o script como **Administrador** para que os atalhos globais funcionem.
